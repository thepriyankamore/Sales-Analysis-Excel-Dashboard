# Sales-Analysis-Excel-Dashboard
Interactive Excel dashboard analyzing product sales, profit, regional performance, and monthly sales trends using Pivot Tables, Charts, KPIs, and Slicers.
Sales Analysis Dashboard using Excel

# Project Overview

This project analyzes product sales performance using Microsoft Excel. The dashboard provides insights into sales, profit, regional performance, and product-wise trends through interactive visualizations.

# Objective
Analyze product sales performance.
Identify high-performing products.
Compare sales across regions.
Evaluate monthly sales trends.
Monitor business KPIs.

# Key Insights
1. Laptop generated the highest sales.
2. South region recorded the highest sales performance.
3. Technology category contributed the highest profit share.
4. January sales exceeded February sales.
5. Total sales reached 237,900 with a profit of 36,650.
6. Interactive slicers allow dynamic analysis by product, region, and category.
